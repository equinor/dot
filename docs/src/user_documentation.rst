.. ### Endpoints

.. #### GET /api/tasks
.. Retrieve all tasks for the authenticated user.

.. **Response:**
.. ```json
.. [
..     {
..     "id": 1,
..     "title": "Task 1",
..     "description": "Description of Task 1"
..     }
.. ]
.. ```
